\# RecruitSmart – AI Candidate Ranking System



\## Project Overview



RecruitSmart is an AI-powered recruitment recommendation system that helps recruiters identify the most suitable candidates for job openings using machine learning and data analytics.



The project integrates SQL, Natural Language Processing (NLP), Machine Learning, FastAPI, and Power BI to build a complete end-to-end recruitment solution.



\## Problem Statement



Recruiters spend significant time manually reviewing resumes. RecruitSmart automates this process by calculating candidate-job relevance scores and ranking applicants based on their skills and experience.



\## Solution



The system preprocesses job descriptions and candidate information, performs feature engineering, stores structured data in SQLite, trains machine learning models to predict match relevance scores, exposes results through FastAPI, and visualizes recruitment insights using Power BI dashboards.

\## Project Outcome



Successfully developed an AI-powered candidate ranking system capable of analyzing recruitment data, predicting candidate relevance scores, and providing recruiter-friendly insights through interactive dashboards and APIs.
