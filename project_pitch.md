\# RecruitSmart Project Pitch



Hello, my name is Chakali Bala Ankanna.



RecruitSmart is an AI-powered candidate ranking system developed to simplify the recruitment process.



The project collects job and candidate information, preprocesses the data, performs feature engineering, stores structured information in an SQLite database, and trains machine learning models to calculate candidate-job match relevance scores.



The recommendation engine is exposed through a FastAPI application, while Power BI dashboards provide recruiters with interactive visualizations such as candidate rankings, skill gaps, and recruitment trends.



This project demonstrates practical skills in Python, SQL, Machine Learning, Natural Language Processing, FastAPI, Power BI, and end-to-end data analytics.
